How it works

  • A GitHub Actions job runs once per day at 08:00 UTC.
  • It renders the UTM Registrar dates page with a headless browser, extracts all academic events, and converts them into all-day calendar entries.
  • The resulting utm.ics file is committed to this repository and served via GitHub Pages.
  • Calendar apps that support webcal / CalDAV will poll the URL periodically and pull in any changes.

Supported campuses: UTM (more coming soon)
